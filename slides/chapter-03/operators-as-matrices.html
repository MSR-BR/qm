Operators as matrices
The observable postulate in a finite-dimensional basis
5 / 15
Guided reading

Once states are vectors, observables become operators acting on those vectors. In a chosen basis, each operator becomes a matrix. The matrix entries tell how the operator maps one basis direction into another.

Postulate for observables and operators

Every measurable quantity is represented by a Hermitian operator. In a basis \(\{|n\rangle\}\), the matrix elements of \(\hat A\) are

\[A_{nm}=\langle n|\hat A|m\rangle.\]

Knowing all matrix elements is equivalent to knowing the operator in that basis.

How the operator acts

If \(|\Psi\rangle=\sum_m C_m|m\rangle\), then

\[\hat A|\Psi\rangle=\sum_{n,m}A_{nm}C_m|n\rangle.\]

The output is another vector. In the Stern-Gerlach problem, applying a spin operator to a spin state tells how that observable is represented before a measurement is interpreted.

Diagonal versus non-diagonal

When an operator is diagonal in a basis, each basis vector is already an eigenstate. For \(\sigma_z\), the \(z\)-basis is diagonal. For \(\sigma_x\), the same \(z\)-basis is not diagonal.

\[\hat\sigma_z=\begin{pmatrix}1&0\\0&-1\end{pmatrix},\qquad \hat\sigma_x=\begin{pmatrix}0&1\\1&0\end{pmatrix}.\]

This is why a beam prepared along \(z\) does not behave as a definite \(x\)-beam.

Practical reading rule

A matrix is not merely a table of numbers. It is a description of a measurement question in a chosen basis. If the basis changes, the matrix representation changes, but the physical predictions do not.

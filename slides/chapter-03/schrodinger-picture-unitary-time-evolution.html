Schrodinger picture: unitary time evolution
States move, operators usually stay fixed
12 / 15
Guided reading

In the Schrodinger picture, the state vector carries time dependence. The rule is the same physical rule as the TDSE of wave mechanics, now written in vector form.

Postulate for time evolution

Between measurements, the state evolves according to the time-dependent Schrodinger equation:

\[i\hbar\frac{d}{dt}|\Psi(t)\rangle=\hat H|\Psi(t)\rangle.\]

If \(\hat H\) is time independent, this evolution can be written with a unitary operator:

\[|\Psi(t)\rangle=\hat U(t)|\Psi(0)\rangle,\qquad \hat U(t)=e^{-i\hat Ht/\hbar}.\]
Why unitary?

Normalization must be preserved, because total probability must remain one:

\[\langle\Psi(t)|\Psi(t)\rangle=\langle\Psi(0)|\hat U^\dagger(t)\hat U(t)|\Psi(0)\rangle=1.\]

This is guaranteed if

\[\hat U^\dagger(t)\hat U(t)=\hat 1.\]
Spin Hamiltonian example

For a spin-1/2 system with \(\hat H=\alpha\hat\sigma_z\), the \(z\)-basis diagonalizes the Hamiltonian. Starting from

\[|\Psi(0)\rangle=\frac{1}{\sqrt2}\left(|+\rangle+|-\rangle\right),\]

the evolved state is

\[|\Psi(t)\rangle=\frac{1}{\sqrt2}\left(e^{-i\alpha t/\hbar}|+\rangle+e^{i\alpha t/\hbar}|-\rangle\right).\]

The probabilities in the \(z\)-basis stay \(1/2\) and \(1/2\), but the relative phase changes. That phase can affect later measurements in another basis.

Measurement connection

Time evolution tells what state arrives at the Stern-Gerlach device. The device then asks a measurement question in a chosen basis and converts the state into outcome probabilities.

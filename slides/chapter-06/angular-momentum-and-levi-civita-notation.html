Angular momentum and Levi-Civita notation
Index notation makes cross products and commutators systematic
2 / 15
Guided reading

The Levi-Civita symbol packages cyclic orientation into one tensor. Together with the Kronecker delta, it turns long component expansions into controlled index contractions.

Component definition
\[\hat L_i=\epsilon_{ijk}\hat x_j\hat p_k\]

Repeated indices are summed over x, y and z. For i = x, only the yz and zy permutations survive, giving \(L_x=yp_z-zp_y\).

Levi-Civita rules
IndicesValue
all distinct, cyclic+1
all distinct, anticyclic-1
any repeated0
Contraction identity
\[\epsilon_{iab}\epsilon_{jcb}=\delta_{ic}\delta_{aj}-\delta_{ij}\delta_{ac}\]

Equivalent index arrangements follow by careful permutations; every swap changes the sign.

Canonical input
\[[\hat x_i,\hat p_j]=i\hbar\delta_{ij},\qquad [\hat x_i,\hat x_j]=[\hat p_i,\hat p_j]=0\]

The delta enforces that only matching Cartesian directions contribute.

Expanding a product commutator

Use \([AB,CD]=A[B,C]D+C[A,D]B\) here because the remaining coordinate-coordinate and momentum-momentum commutators vanish. Substitution leaves terms proportional to \(x_ap_d-x_dp_a\), which can be recognized as another angular component.

Dimensional and sign check

Every angular-momentum commutator must have dimensions of angular momentum squared. The right side contains ħ times L, so the units match. Cyclic order x→y→z gives a plus sign; reversing two labels gives a minus sign.

Algebra-to-physics audit

Before accepting an angular-momentum calculation, verify four layers. Operator layer: preserve order and use the correct commutator sign. Spectrum layer: confirm that m changes only in unit steps and remains between -l and +l. Representation layer: state the basis order before reading matrix rows or columns, and check Hermiticity for observables. Physics layer: distinguish an exact eigenvalue from an expectation value or variance. A zero transverse mean does not imply a zero transverse measurement; a basis rotation does not change the underlying observable; and a semiclassical vector cone does not assign simultaneous sharp values to all three components. These checks connect the formal algebra to what an actual sequence of measurements can reveal.

Why learn the notation

Levi-Civita notation is not cosmetic. It makes rotational structure visible and lets the same derivation work for any component without repeating three separate calculations.
